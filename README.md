## 🔄 Weekly routine (for the admin / custodian)

### Every day (next morning after entries come in):
1. Open `tpl_config.html` on your laptop
2. Upload the MS Forms Excel export in the **Daily Entries** section
3. Click **⬇️ Download data.json**
4. Go to this GitHub repo → click `data.json` → click the ✏️ pencil icon
5. Delete the old content, paste the new file content (or use "Upload files")
6. Click **Commit changes**
7. Wait ~1 minute → dashboard updates for everyone ✅

### When store mapping changes (new store, closed store, ROM change):
1. Update your master Excel file (`Store Name | Store Code | ROM Name`)
2. Open `tpl_config.html` → upload the file in **Base Store Data**
3. Review the preview → click **Apply**
4. Click **⬇️ Download data.json** → upload to GitHub (same steps above)

### When there's a holiday or excluded day:
1. Open `tpl_config.html` → add the date in **Exclude TPL Days**
2. Download data.json → upload to GitHub
